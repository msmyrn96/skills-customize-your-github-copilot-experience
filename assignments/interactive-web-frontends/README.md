# 📘 Assignment: Interactive Web Frontends with JavaScript

## 🎯 Objective

Create dynamic web pages that interact with REST APIs using HTML, CSS, and JavaScript. You'll learn to fetch data from APIs, handle user interactions, and update the page content dynamically.

## 📝 Tasks

### 🛠️ Build the Page Structure

#### Description
Create the basic HTML structure and CSS styling for a web page that will display and manage data from a REST API.

#### Requirements
Completed program should:

- Create a complete HTML document with proper semantic structure
- Include CSS for attractive styling and responsive layout
- Add form elements for user input (text fields, buttons)
- Create containers for displaying API data

### 🛠️ Add JavaScript Interactivity

#### Description
Implement JavaScript functionality to handle user interactions and update the page dynamically without full page reloads.

#### Requirements
Completed program should:

- Add event listeners for button clicks and form submissions
- Implement functions to update page content dynamically
- Use DOM manipulation to show/hide elements and update text
- Handle user input validation on the frontend

### 🛠️ Connect to REST API

#### Description
Integrate with a REST API using JavaScript's fetch API to send requests and handle responses.

#### Requirements
Completed program should:

- Use fetch() to make GET requests to retrieve data
- Implement POST requests to send data to the API
- Handle API responses and update the UI accordingly
- Display appropriate loading states and error messages
